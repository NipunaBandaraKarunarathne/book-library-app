How to Clone and run the Library system
===========================================

1. Clone the Project.

git clone https://github.com/NipunaBandaraKarunarathne/book-library-app.git

2. go into that location using cd/ command and install packages

npm i


3. run the application and json-server using following two commands. 

npm run dev

npm run mock  


4. open you browser, past this in url and enter,

http://localhost:3000/
